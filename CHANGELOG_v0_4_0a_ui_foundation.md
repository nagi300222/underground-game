# CHANGELOG v0.4.0a-ui-foundation-draft

目的：v0.4.0 UI大改修の第1段階として、既存viewキー・class/id・render全置換前提を維持したまま、入口とホーム導線を整理。

## 変更
- VERSION / SAVE_VERSION / index title / sw cache を `v0.4.0a-ui-foundation-draft` に更新。
- ナビを9入口から5タブへ再配置。
  - ホーム / 活動 / 予定 / 携帯 / バンド。
  - `songs`、`shop`、`library`、`bandbook`、`log`、`dev` の旧viewキーは削除せず、ホーム内の文脈ボタンから到達可能。
- ナビのactive表示に親カテゴリマッピングを追加。
- ホーム画面をL1主目的中心に再構成。
  - 次にやることカード。
  - 最大面積の「今週の行動」ボタン。
  - 予定ミニライン。
  - 最新ログ折りたたみ。
  - 管理メニュー小タイル。
- L0ヘッダー相当として、上部ステータスにターン/山場/未読/疲労顔アイコンを追加。
- CSSトークンを追加し、v0.4系の見た目差し替えの土台を作成。

## 変更しないもの
- ライブ準備ステップ化はまだ未実装。v0.4.0bで実施予定。
- `render()` の全置換方式は維持。
- `bindEvents()` の既存class/idは維持。
- GRAND条件、ターン進行、メール返信、サポート、打ち上げ、通常行動ミニイベント、バンドイベント、スキル効果は変更なし。
