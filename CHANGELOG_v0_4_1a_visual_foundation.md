# CHANGELOG v0.4.1a-visual-foundation-draft

## 目的
v0.4.0-rc3を土台に、現行機能を壊さずにv0.4.1デザイン方針を先行反映した版。

## 追加・変更

- 右下固定の「📱 携帯」ボタンを追加。
  - どの通常画面から押しても携帯トップへ移動。
  - `phoneSubView = "menu"` / `activeMailId = null` に戻す。
  - 未読がある場合はバッジ表示。
- ライブ本番中でも、携帯を開いた場合は携帯画面を表示できるようにした。
  - ホームへ戻る操作でライブ準備画面へ戻る。
  - ライブ準備DOM自体は変更なし。
- 上部ステータスを再整理。
  - ターン、資金、ファン、疲労を見やすく表示。
  - 次予定、知名度、業界評価、未読をサブ情報として表示。
- 一列の「週の流れ」レールを追加/再整理。
  - 現在ターン、近いターン、近いライブ、UNDER/GRANDを横一列で表示。
  - クリックで予定画面へ移動。
- ホームをやや圧縮。
  - 主ボタンを「今週どうする？」へ変更。
  - 予定ミニラインを「週の流れ」へ変更。
  - メニューは「よく使う」と「ほかのメニュー」に分離。
- 文言をやや自然な日本語へ調整。
  - 英語見出し/AIっぽい説明文を増やさず、短いゲーム内文言へ寄せた。
- CSSで可読性と折り返しを調整。
  - 1〜2文字だけの不自然な改行を避けるため、主要ボタン/タブ/ステータスは `white-space: nowrap` + `text-overflow` 寄りに調整。
  - スマホ幅で資金/ファン/疲労が読めるようにフォントサイズを `clamp()` 化。
- GitHub Pages用に `.nojekyll` を同梱。
