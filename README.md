# アンダーグラウンド（仮） v0.3.7

HTML/CSS/JavaScript製のスマホ縦画面向けプロトタイプです。

## v0.3.7 修正内容

- スケジュール予約候補カードの会場費/ドル表示まわりの幅を調整
- スケジュール横スクロールの初期位置を、現在ターンが見える位置へ自動補正
- ライブ準備のセトリ選択をプルダウンからカード式の別メニューに変更
- 5曲セトリの各枠に「曲を選ぶ」「詳細」ボタンを追加
- 曲選択メニュー内で曲ステータス・タグ・棒グラフを確認可能に変更
- ライブ準備のメンバー行に「詳細」ボタンを追加
- 曲/メンバー詳細をモーダル表示できるように追加
- 詳細確認や曲選択を開いても、選択中のセトリ/物販/サポート/担当楽器が残るよう調整

## アップロード方法

ZIPを解凍して、中身の `index.html`, `game.js`, `style.css`, `manifest.json`, `sw.js`, `README.md`, `VERSION.txt` をGitHubリポジトリ直下へ上書きしてください。

`VERSION.txt` が `underground_game_v0.3.7` になっていれば反映済みです。

古い表示が残る場合は、ゲーム内の「最新版を読み込む」を押してください。v0.3.7では `sw.js` のキャッシュ名も更新しています。
