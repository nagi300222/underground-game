# アンダーグラウンド（仮） v0.3.4

スマホ縦画面向けのバンド育成プロトタイプです。v0.3.4では、導入・ライブ・行動結果・曲作り・予約まわりの演出とUIを調整しました。

## 主な変更

- 作詞作曲せずに行動した時の確認ポップアップを見やすく調整
- ターン終了後の `○/30ターン / NEXTライブ` を中央に大きく表示
- バンド名決定イベントを、初ライブ前ターンの行動後に変更
- ライブ中演出をリアルタイム風に変更
  - 1曲目〜5曲目を約1秒ごとに順番表示
  - 進捗バーが曲ごとに進む
  - 各曲終わりに、盛り上がり・ひらめき・疲労などのアイコン表示
- 初ライブ後にスケジュール帳チュートリアルを追加
- ライブ予約時に、会場費・キャパ・準備要求つきの確認ポップを表示
- 練習後、各キャラのステータス上昇を表示
- 疲労表示を整数%に変更
- ポップアップを中央表示に調整
- 曲完成・曲強化・メンバー加入のポップアップに棒グラフ表示を追加
- ホームに戻るボタンを左下の丸ボタンに変更
- ガレージ閉鎖画面にもホームボタンを追加
- バンド編成でキャラをタップすると詳細表示
- 初ライブ後、主人公がボーカル以外の楽器も担当可能になった演出を追加
- 直近7ターン以内にライブ予定がない場合のリマインドを追加
- ライブ後、ファンから差し入れで疲労回復ドリンクをもらうイベントを追加

## 起動方法

PCなら `index.html` をブラウザで開くだけでも動きます。スマホ確認はGitHub Pages推奨です。

## GitHub Pages更新

ZIPを解凍して、中身のファイルをリポジトリ直下に上書きアップロードしてください。ZIP自体ではなく、`index.html` / `game.js` / `style.css` などをアップロードします。

## 更新時の注意

古い表示が残る場合は、ゲーム内の「最新版を読み込む」を押してください。v0.3.4では `sw.js` のキャッシュ名も更新しています。
