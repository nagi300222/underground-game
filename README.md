# アンダーグラウンド（仮） v0.3.28

v0.3.27をベースに、他バンド図鑑・対バン候補・ライブ評価バランス・打ち上げイベントを強化した版。

## 主な変更

- ほかのバンドを6組から20組へ増加
  - 序盤の地元バンド、中盤の格上、終盤の有名バンドまで段階化
  - 各バンドに知名度/集客力/ジャンル/レベル/メモを設定
- 対バン候補の選出を調整
  - 自分たちの人気・業界評価・信頼・交流度で、呼べる/呼ばれる候補が変化
  - 交流が低すぎる相手は候補に出にくい
  - 交流が高いと格上も候補に入りやすい
- 有名バンドとの共演成長を追加
  - 対バン/ブッキングで格上バンドと共演すると、メンバーの技術/感性/カリスマが伸びやすい
  - 有名な相手ほど上がり幅が大きい
- ライブ評価・コピー曲バランスを再調整
  - コピー曲の基礎能力を下方修正
  - コピー曲入りセトリのボーナスを抑制
  - コピー曲が多いほど表現/戦略/最終評価が伸びにくい
  - S評価ラインを少し上げ、コピー曲中心でSが出にくいようにした
  - ワンマンライブの集客要求をさらに上げた
- 打ち上げイベントを追加/強化
  - 他バンド込みの打ち上げ時のみ交流/紹介/追加成長が発生
  - 連絡先交換、機材トーク、曲作り相談、紹介の話、ライブハウス情報、格上からの助言などのイベントを追加
  - イベント内容と上昇ステータスが結果ポップに表示される
  - 紹介イベントでメンバー候補メールが届く場合がある
- ライブ結果ログに、格上対バン刺激の情報を追加

## 確認ポイント

1. ほかのバンド情報/図鑑に20組表示されるか
2. 序盤で呼べるバンドが低〜中レベル中心になっているか
3. 成長後、格上バンドが候補に入りやすくなるか
4. コピー曲中心のセトリでSが出すぎないか
5. ワンマンライブが以前より難しくなっているか
6. 他バンド込みライブ後の打ち上げで、イベント名/説明/ステータス上昇が見えるか
7. 紹介イベントでメンバー候補メールが届くことがあるか
8. ライブ結果ログに、共演による成長情報が残るか
