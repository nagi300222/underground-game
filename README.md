# アンダーグラウンド（仮） v0.3.10

HTML/CSS/JavaScript製のスマホ縦画面向けプロトタイプです。

## v0.3.10 修正内容

- タイトル画面を追加しました。
  - はじめから
  - つづきから
- セーブスロットを2つに増やしました。
- セーブ/ロード時にスロット選択画面を表示するようにしました。
- 「はじめから」で既存セーブが即消えないよう、スロット選択と確認を追加しました。
- エンディング後に、以下を選べるようにしました。
  - このまま続ける
  - はじめから選ぶ
- スマホでボタンを押したあと、黄色い枠が残って誘導表示に見える問題を軽減しました。
- v0.3.9の追加メンバー、セトリボーナス、マンネリ、ライブアレンジ、50ターン化は継続しています。

## アップロード方法

ZIPを解凍して、中身の `index.html`, `game.js`, `style.css`, `manifest.json`, `sw.js`, `README.md`, `VERSION.txt` をGitHubリポジトリ直下へ上書きしてください。

`VERSION.txt` が `underground_game_v0.3.10` になっていれば反映済みです。

古い表示が残る場合は、ゲーム内の「最新版を読み込む」を押してください。v0.3.10では `sw.js` のキャッシュ名も更新しています。
