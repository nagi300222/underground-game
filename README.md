# アンダーグラウンド（仮） v0.3.25

2ターン目チュートリアル詰み・疲労不足時の曲作成破綻・コピー曲バランス調整版。

- 2ターン目はホームから開始し、1ターン目の未完成曲だけを完成誘導するように修正
- 2ターン目に別の未完成曲を作っても、チュートリアルが再ロックされないように対象ID固定化
- 疲労不足時は作詞・作曲・曲強化・曲編集の次ページへ進まず、未完成曲だけ残る事故を防止
- 曲Ptありの説明を「曲の強化が可能」に変更
- 状況メモの「必要時だけ表示」バッジを削除
- 歌詞ワードのランダム候補と判定辞書を大幅増量
- コピー曲の基礎能力とライブ上のコピー曲補正を下げ、コピー曲依存でSが出すぎないよう調整

GitHub PagesではZIPの中身をリポジトリ直下へ上書きしてください。
