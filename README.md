# アンダーグラウンド（仮） v0.3.23

控えめガイドUI / 安全目安システム版。

- 常時おすすめで埋めず、ホームは短い状況メモと色タグ中心へ調整
- ライブ候補に「安全寄り / 挑戦圏 / 背伸び / 危険」タグを追加
- ライブ候補のタグは最適解ではなく、破綻しにくい安全目安として表示
- 物販に安全目安数を追加。ただし最高利益の理論値より低めに表示
- 物販の現在仕入れ数に応じて「控えめ / 安全目安 / 攻め / 売残注意」を表示
- スケジュール初回に、安全目安は正解ではない旨を一度だけ説明
- UI文言を「おすすめ」より「状況メモ」「安全目安」寄りに調整

GitHub PagesではZIPの中身をリポジトリ直下へ上書きしてください。
