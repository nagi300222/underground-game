# アンダーグラウンド（仮） v0.2.3

インディーズバンド育成シミュレーションADVのスマホ縦画面向けプロトタイプです。

## v0.2.3 変更点

- 画面遷移アニメを追加
- テロップ演出を強化
- コマンド後にランダムイベントポップアップを追加
- 作詞作曲で高完成度が出た時のポップアップを追加
- 新曲完成ポップアップを追加
- メンバーカードにタップ/ホバー演出を追加
- ライブ結果専用画面を追加
- v0.2.2のスマホ縦画面UI、PWA、セーブ/ロード、最新版読み込みボタンは維持

## 起動方法

PCなら `index.html` をブラウザで開くだけでも動きます。
スマホ実機確認は GitHub Pages 推奨です。

## GitHub Pagesへの反映

ZIPを解凍して、中身をリポジトリ直下へ上書きアップロードしてください。

必要な主なファイル：

```txt
index.html
style.css
game.js
manifest.json
sw.js
icon-192.png
icon-512.png
README.md
VERSION.txt
GITHUB_PAGES_GUIDE.md
```

アップロード後は、同じPages URLで更新されます。古い表示が残る場合は、ゲーム内の「最新版」または「最新版を読み込む」を押してください。

## 注意

v0.2.3は演出追加が中心です。イベント内容やライブ結果画面の見た目はまだ仮です。
次に調整するなら、イベント種類の追加、ライブ結果画面のテンポ、メンバーカード詳細UIの強化が候補です。
