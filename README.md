# アンダーグラウンド（仮） v0.3.27

ライブ/携帯/SNS/他バンド情報の拡張版。v0.3.26のライブ土台を、実際に遊ぶ導線に寄せて整理。

## 主な変更

- 初ライブを「ライブハウスイベント参加」扱いに変更
  - 初ライブから他バンド込みの打ち上げを体験可能
  - 打ち上げは強制ではない
- スケジュール帳を拡張
  - 予定済みライブをライブスケジュールとして表示
  - 自主企画候補とは別に、ライブハウスイベント候補を表示
  - ライブハウスイベントは自分から参加可能
- ワンマンライブの難易度を上げた
  - 自分たちだけの集客不足がより強く響く
- セトリ自動セット追加
  - 新曲・マンネリ回避を優先
  - コピー曲は曲数不足時のみ補充
  - 同じ曲は重複しない
  - 曲順/ボーナスは考慮しない
- メールUIを変更
  - 件名と送信者だけ表示
  - 押すと本文が開き、既読になる
  - 出演通知はメール本文から参加可能
- 携帯に未読バッジを追加
- SNSを携帯メニューから起動する形に変更
  - 雑多な一般人投稿、他バンド告知、流行投稿が流れる
  - 投稿ボタン追加：告知/雑談
  - 投稿は1日1回
  - 投稿10件で主人公スキル「SNS上手」習得
  - SNS上手で宣伝効果アップ
- 打ち上げ仕様を調整
  - 交流/紹介/ステータスアップは、他バンド込みの打ち上げ時のみ
  - 打ち上げ結果で交流・ステータス上昇・二日酔い判定を表示
  - 二日酔いメールは廃止
- バンド編成を「バンド情報」に変更
  - 自身のバンド情報
  - バンド名変更機能追加（認知度が少し減少）
  - ほかのバンド情報を表示
- 図鑑にほかのバンド図鑑を追加
- ライブ診断ログを廃止し、過去のライブ結果をログ/携帯/スケジュールで見られるように変更
- 曲作りをしていない時の行動前確認ポップを廃止
- フェス条件ポップの頻度を低下
  - 基本はフェス5ターン前、または条件達成時
- ポップを最大3件までまとめて表示

## 確認ポイント

1. 初ライブが「ライブハウスイベント」として表示されるか
2. 初ライブ後、打ち上げで他バンド交流/ステータスアップが分かるか
3. スケジュール帳にライブハウスイベント候補と自主企画候補が分かれて出るか
4. メールが件名/送信者表示になり、押すと既読になるか
5. 携帯/タブに未読数字バッジが出るか
6. SNSを携帯から起動し、告知/雑談投稿ができるか
7. セトリ自動セットでコピー曲が不足時だけ入るか
8. ワンマンの成功難易度が上がっているか
9. バンド情報で改名と他バンド情報が見られるか
