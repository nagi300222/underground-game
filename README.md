# アンダーグラウンド（仮） v0.3.26

ライブ仕様の拡張土台版。完成形まで一気に作り込まず、追加後にどこが悪いか追える診断ログ付きで整理。

- ライブ種別を追加
  - 自主企画：ワンマン
  - 自主企画：対バン企画
  - ブッキング：ライブハウスイベント
  - ブッキング：他バンド企画
- スケジュール帳の予約候補を、自主企画ライブ専用に変更
- 会場候補を毎ターン固定リストではなくランダム寄りに生成
- ブッキング通知はホームの「携帯」→メールから参加する形式に追加
- 携帯にメール/SNS/ライブ診断ログを追加
- ライブ結果にライブ種別、自分目当て客、他バンド目当て客、費用種別を反映
- ワンマンはハイリスク・高成長、対バン/ブッキングは交流と安定寄りに分離
- 全曲コピー曲、同じ曲偏重、同一セトリ連発で交流注意が出るように追加
- ライブ後の打ち上げを追加
  - 疲労80%以下で参加可能
  - 参加で疲労+20%、交流/メンバー成長/紹介イベント
  - 1〜20%で二日酔い。発生時は次ターン休憩のみ
- 宣伝でブッキング招待率が上がるように強化
- 主人公スキル「段取り上手」を追加
  - 練習/休憩/バイト/募集/宣伝/会話/曲作り/ライブを各経験後、疲労増加0.8倍

確認ポイント：

1. 初ライブ後、スケジュール帳に自主企画候補が出るか
2. 携帯メールにブッキング通知が届くか
3. ブッキング参加後、予定に追加されるか
4. ライブ結果の集客が「自分目当て/他バンド目当て」に分かれるか
5. 携帯のライブ診断ログで、評価や収支の原因が追えるか
6. 打ち上げ参加後、疲労・交流・二日酔いが自然に動くか

GitHub PagesではZIPの中身をリポジトリ直下へ上書きしてください。
