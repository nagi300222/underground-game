# アンダーグラウンド（仮） v0.2.2

スマホ実機確認に寄せた PWA / セーブ対応版です。

## 主な追加

- v0.1.9 のライブスケジュール管理を維持
- `manifest.json` を追加
- `sw.js` を追加
- PWAアイコンを追加
- ホーム画面追加を想定したメタタグを追加
- 手動セーブ / ロード / セーブ削除を追加
- 行動・曲作り・ライブ後にオートセーブ
- スマホ横画面向けの調整を追加
- GitHub Pages公開手順書 `GITHUB_PAGES_GUIDE.md` を追加

## 起動方法

PCなら `index.html` をブラウザで開くだけでも動きます。
スマホ確認は GitHub Pages 推奨です。

## ファイル構成

```txt
index.html
style.css
game.js
manifest.json
sw.js
icon-192.png
icon-512.png
VERSION.txt
GITHUB_PAGES_GUIDE.md
```

## セーブについて

ブラウザの `localStorage` に保存しています。
同じ端末・同じブラウザ・同じURLならロードできます。
URLやブラウザを変えると別データ扱いになる場合があります。

## 注意

Service Worker のキャッシュにより、更新後も古い表示が残ることがあります。
その場合はスマホ/PCのブラウザで再読み込み、またはサイトデータ削除を試してください。


## v0.2.2 変更点

- スマホ横画面で文字・ボタン・カード余白を縮小
- 育成スケジュールを1列の横スクロール表示に変更
- ライブスケジュール予定も1列の横スクロール表示に調整
- タブ・コマンド・ホーム画面の情報密度を上げた
- PWA、セーブ、ロード、GitHub Pages公開対応は継続

スマホで古い表示が残る場合は、ホーム画面のアイコンからではなく一度ブラウザで開き直すか、ブラウザのサイトデータ/キャッシュを削除してください。


## v0.2.2 変更メモ

- スマホ縦画面で見やすいUIに調整。
- スケジュールと育成タイムラインは横スクロールのまま、画面全体は縦スクロール中心に変更。
- ヘッダーとスマホ確認パネルに「最新版を読み込む」ボタンを追加。
- ボタンを押すとService Workerとアプリキャッシュを削除し、セーブデータを残したまま再読み込みします。
- Service Workerをネットワーク優先に変更し、GitHub Pages上の更新反映を優先。

※ v0.2.1以前が強くキャッシュされているスマホでは、初回だけブラウザ再読み込みまたはサイトデータ削除が必要な場合があります。v0.2.2以降はゲーム内ボタンで更新できます。
