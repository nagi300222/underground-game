# アンダーグラウンド（仮） v0.3.15

## v0.3.15 修正内容

- 物販の仕入れ数UIを、ダイヤル式錠風の3桁UIへ変更。
- 百の位・十の位・一の位をそれぞれ0〜9から選択して数量を決められるようにしました。
- 仕入れ数は `000`〜`999` の大きな表示で確認できます。
- 微調整用に `−1 / ＋1` ボタンも残しています。
- 数量変更時に物販リストが一番上へ戻らないよう、再描画なしで仕入れ総額・所持金・差額だけ更新します。

## アップロード

ZIPを解凍して、中身の `index.html`, `game.js`, `style.css`, `manifest.json`, `sw.js`, `README.md`, `VERSION.txt` をGitHubリポジトリ直下へ上書きしてください。

`VERSION.txt` が `underground_game_v0.3.15` になっていれば反映済みです。

古い表示が残る場合は、ゲーム内の「最新版を読み込む」を押してください。v0.3.15では `sw.js` のキャッシュ名も更新しています。
