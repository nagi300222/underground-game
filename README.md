# アンダーグラウンド（仮） v0.3.6

スマホ縦画面向けのバンド育成プロトタイプです。

## v0.3.6 修正内容

- ライブ演出が5曲目終了後に自動で結果へ進まないよう変更
- LIVE COMPLETE表示で止まり、「結果を見る」ボタンを押して結果へ進む方式に変更
- 1曲目と5曲目が同じ曲の場合など、同一曲の影響表示を実際に影響が出た曲へ移動
- 各曲のライブ演出コメントを曲順・タグ・ジャンルに応じて分岐
- 疲労80%超、資金マイナス時にステータス表示を赤くする警告を追加
- 7ターン以内にライブ予定がない場合、「ライブの予定をいれないとな……」ポップを追加
- 2回目以降のライブ後に、UNDER FES目標まで何が足りないかを表示するポップを追加

## アップロード方法

ZIPを解凍し、中身のファイルをGitHubリポジトリ直下に上書きアップロードしてください。
`VERSION.txt` が `underground_game_v0.3.6` になっていれば反映済みです。

古い表示が残る場合は、ゲーム内の「最新版を読み込む」を押してください。v0.3.6では `sw.js` のキャッシュ名も更新しています。
