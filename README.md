# アンダーグラウンド（仮） v0.2.5

スマホ縦画面での操作感をさらに整理した版です。

## 主な変更

- 曲エディタの「戻る」を、メニュー戻りではなく **一つ前の画面へ戻る** に変更
- ホームメニューに **ショップ / ログ** を追加
- 画面端に **ホームへ戻る** フローティングボタンを追加
- スマホ下部固定フッター風タブを解除し、画面を圧迫しにくい表示へ調整
- 資金・ファン・知名度・業界評価・疲労・編成人数の表示をコンパクト化
- ヘッダーの「予定」からスケジュール帳を開けるように変更
- ライブ予約をプルダウンではなく、**スケジュール帳の候補カードを選ぶ方式**に変更
- 残り2ターン以内のライブは予約・通常キャンセル不可
- ドタキャン時は **会場費＋損料** を支払う
- v0.2.4までのセーブ/ロード、PWA、イベント演出、ライブ結果画面は維持

## 起動方法

PCなら `index.html` をブラウザで開くだけでも動きます。スマホ確認はGitHub Pages推奨です。

## ファイル構成

```txt
index.html
style.css
game.js
manifest.json
sw.js
icon-192.png
icon-512.png
VERSION.txt
GITHUB_PAGES_GUIDE.md
```

## 更新時の注意

GitHub PagesへはZIPそのものではなく、中身のファイルをすべて上書きアップロードしてください。
古い表示が残る場合は、ゲーム内の「最新版」ボタンを押してください。
