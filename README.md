# アンダーグラウンド（仮） v0.2.7

曲作りのジャンル生成を、メインジャンル2択＋派生抽選方式に変更した版です。

## 主な変更

- 新曲作成時、メインジャンルを2つ選択する方式に変更
  - 例：ロック → ポップ
  - 例：ロック → ロック
- 同じメインジャンルを2回選ぶと、そのジャンル狙いになる
  - 基本：メインジャンル 70%
  - 派生サブジャンル 30%
- 異なるメインジャンルを選ぶと、どちらかのメインジャンルが生成される
  - 基本：1つ目 35% / 2つ目 35% / 派生 30%
- 相性の良い組み合わせは、狙い派生が強くなる
  - 例：パンク → ポップ でポップパンクを狙える
  - 成功組み合わせは派生 50% / メイン側 50%
- サブジャンルは直接選ぶものではなく、メインジャンルの組み合わせから生成される派生として扱う
- 曲エディタの確認欄に、狙いジャンルと確率メモを表示
- 制作中曲にも「狙い：パンク→ポップ」のような情報を保持

## 仮の組み合わせ表

```txt
パンク + ポップ -> ポップパンク
ロック + ポップ -> ポップロック
ロック + パンク -> ギターロック
ロック + ヒップホップ -> ラップロック
ロック + メタル -> ハードロック
メタル + パンク -> メタルコア
ジャズ + ポップ -> ジャズポップ
ジャズ + ヒップホップ -> ジャズラップ
クラシック + メタル -> シンフォニックメタル
ヒップホップ + ポップ -> ラップポップ
```

この表は仮です。今後、曲ジャンルの気持ちよさに合わせて追加・修正できます。

## 起動方法

PCなら `index.html` をブラウザで開くだけでも動きます。スマホ確認はGitHub Pages推奨です。

## ファイル構成

```txt
index.html
style.css
game.js
manifest.json
sw.js
icon-192.png
icon-512.png
VERSION.txt
GITHUB_PAGES_GUIDE.md
```

## 更新時の注意

GitHub Pagesに上書きしたあと古い表示が残る場合は、ゲーム内の「最新版を読み込む」を押してください。  
それでも古い場合は、スマホ側のサイトデータやService Workerキャッシュを削除してください。
