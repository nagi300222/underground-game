# アンダーグラウンド（仮） v0.3.0

スマホ縦画面向けのバンド育成プロトタイプです。v0.3.0では、開始イントロ、初回チュートリアル、行動結果演出、ライブ進行演出、加入・発見・曲完成演出を追加しました。

## 主な変更

- ゲーム開始時に UNDER FES ポスター風イントロを追加
  - 「必ず、UNDERフェスに出てやる！」
  - 30ターン以内に曲・仲間・ライブ実績を作る目標表示
- 1ターン目チュートリアルを追加
  - 最初は作詞・作曲だけ操作可能
  - 曲作り後に今週の行動へ誘導
  - 行動後に「5ターン目に初ライブだ！」演出
- 初ライブ前ターンにバンド名決定イベントを追加
- 初ライブ後に謎の人物イベントを追加
  - 次ターンに UNDER FES 主催関係者と判明
  - フェス条件情報を開示
- 行動後にライブ結果風の演出を追加
  - 資金 / 知名度 / 疲労 / 信頼度などの変化を表示
  - 行動に合わせた小イベントを表示
- 作詞・作曲の50% / 100%演出を強化
  - 作詞50%完成 → 次は作曲をしよう
  - 作曲50%完成 → 次は作詞をしよう
  - 新曲完成時にジャンル・タグ表示
- メンバー加入演出を追加
  - 本加入 / 仮加入で表示を分ける
- ジャンル発見演出を追加
  - 初発見ジャンルを図鑑登録風に表示
  - レアジャンルは派手めに表示
- ライブ中演出を追加
  - ライブ進捗バー
  - 曲ごとのテロップ
  - ひらめき / アンコール / 盛り上がり / 疲労などのアイコン表示
  - バーMAX後に結果画面へ
- ライブ結果後にファン・知名度・業界評価・資金の反映演出を追加
- v0.2.8のジャンル派生・アレンジ条件・曲作り確認仕様は維持

## 起動方法

1. ZIPを解凍
2. `index.html` をブラウザで開く
3. GitHub Pagesに置く場合は、ZIPではなく中身のファイルをリポジトリ直下にアップロード

## GitHub Pages 更新時の注意

- `index.html`, `game.js`, `style.css`, `manifest.json`, `sw.js`, `icon-192.png`, `icon-512.png`, `README.md`, `VERSION.txt` を上書きアップロード
- 表示が古い場合はゲーム内の「最新版」ボタン、またはブラウザのサイトデータ削除
- `VERSION.txt` を開いて `v0.3.0` と表示されればPages側は更新済み

## 今後の差し替えポイント

- イントロの仮ポスターは後で画像素材に差し替え可能
- 行動結果イベントは `maybeGenerateCommandEvent` に追加
- ライブ中テロップは `liveSongLine` で調整
- ジャンル発見・レア演出は `registerGenreDiscovery` で調整
- チュートリアル進行は `tutorialStage` で管理
