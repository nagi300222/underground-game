# アンダーグラウンド（仮） v0.3.16

## v0.3.16 修正内容

- 30ターン目を中間目標 `UNDER FES`、50ターン目を最終目標 `GRAND UNDER FES` に変更。
- 後半目標の条件を強化し、ファン・知名度・業界評価・代表/定番曲・後半新曲を要求する形に調整。
- 物販収入を下方修正。売れ残り買い取りを仕入れ値35%へ変更し、売れやすさも控えめに調整。
- ショップアイテム価格を上げ、ドリンク/機材/エフェクトに所持上限を追加。
- ドリンク回復量を下げ、休憩コマンドの意味が残るように調整。
- 疲労が高い時、練習・宣伝の効率が下がるように変更。
- 曲作りを1ターン1回固定から、疲労が100%を超えない限り複数回できる形へ調整。
- 作詞/作曲/強化で疲労が増え、作詞作曲Lv・ひらめきで疲労軽減/無料化が起こる土台を追加。
- 曲完成前に曲名・テーマを確認/編集できる画面を追加。
- 作詞キーワード入力を3語入力式へ変更。
- 曲の使用回数、連続使用回数、前回何曲目だったかを曲詳細やセトリ選択で表示。
- ジャンル発見、レアジャンル発見、定番曲化のポップ表示を追加/強化。
- 練習や会話イベントで、次回作詞/作曲が疲労なしになる「ひらめき」イベントを追加。

## アップロード

ZIPを解凍して、中身の `index.html`, `game.js`, `style.css`, `manifest.json`, `sw.js`, `README.md`, `VERSION.txt` をGitHubリポジトリ直下へ上書きしてください。

`VERSION.txt` が `underground_game_v0.3.16` になっていれば反映済みです。

古い表示が残る場合は、ゲーム内の「最新版を読み込む」を押してください。v0.3.16では `sw.js` のキャッシュ名も更新しています。
