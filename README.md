# アンダーグラウンド（仮） v0.3.29

v0.3.28をベースに、プレイテストで出たテンポ低下・曲強化の面倒さ・ライブ予定消失・最終フェス準備画面の不具合を修正した版。

## 主な変更

- スケジュール予約済みライブが、別日程のブッキング/ライブハウスイベント参加時に消えないよう、ライブ予定追加処理を保護
- 曲作成のテーマ初期値を「---」に変更
- 練習コマンドの結果OK後に次ターンへ進む導線を明示
- ライブポジションに「ギターボーカル」「ベースボーカル」を追加
- 「その他楽器」表記を「ピアノなど」に変更
- 曲Ptを廃止
- 曲強化を「編曲→曲強化→曲一覧→曲を選ぶ→確認ポップ→強化」に変更
- 曲に強化Lvを追加
- 曲作り/曲強化は1ターン1回までに制限し、テンポを改善
- GRAND UNDER FESのセトリ自動セット/曲変更でホームに戻される不具合を修正
- 主人公スキル一覧に表示/非表示タブを追加

## 確認ポイント

- 予約済み自主企画ライブと、別日程のライブハウスイベントが両方スケジュールに残るか
- 曲作成の最終確認でテーマが「---」初期表示になっているか
- 練習結果のOK後にターンが進むか
- ギターボーカル/ベースボーカルを選んでライブできるか
- 曲強化がポイント振り分けなしで進むか
- 最終フェス当日のセトリ自動セット/曲変更後もライブ準備画面に残るか
