# アンダーグラウンド（仮） v0.3.14

## v0.3.14 修正内容

- 物販の仕入れ数UIを調整。
- `+5 / -5` ではなく、`− / ＋`、大きな数量表示、横スライダー、直接入力で操作できるように変更。
- 数量変更時にライブ準備画面全体を再描画しないようにし、物販リストが一番上に戻る問題を修正。
- 仕入れ総額、所持金、差額はその場で更新。
- スマホ操作時に数量欄が狭くて見えない問題を軽減。

## アップロード

ZIPを解凍して、中身の `index.html`, `game.js`, `style.css`, `manifest.json`, `sw.js`, `README.md`, `VERSION.txt` をGitHubリポジトリ直下へ上書きしてください。

`VERSION.txt` が `underground_game_v0.3.14` になっていれば反映済みです。

古い表示が残る場合は、ゲーム内の「最新版を読み込む」を押してください。v0.3.14では `sw.js` のキャッシュ名も更新しています。
