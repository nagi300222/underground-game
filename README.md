# アンダーグラウンド（仮） v0.2.4

曲エディタ画面遷移＋ホーム簡素化版。
スマホ縦画面で、プルダウン中心ではなく「メニューを進めて選ぶ」操作に寄せています。

## 主な変更

- 作詞・作曲を **曲エディタメニュー** 化
- 新曲 / 曲強化 / 未完成曲を最初に選ぶ画面を追加
- 新曲作成は、曲名 → 作詞/作曲 → メインジャンル → サブジャンル → テーマ → キーワード → アレンジ → 担当メンバー の順で進行
- 曲強化は、曲選択 → 強化カテゴリ → 担当メンバー の順で進行
- 未完成曲は、制作中の曲一覧 → 未作成の作詞/作曲 → 担当メンバー の順で進行
- ホームを簡素化
  - 上から、ログ / おすすめ行動・Tips / 今週やること
  - セーブ・ロード・最新版ボタンはヘッダー側に集約
- フェス条件は最初から表示せず、ライブ後イベントで情報入手する方式に変更
- スケジュール表示をコンパクト化
  - 左上に現在ターン
  - 右上にNEXTライブと残りターン
- スケジュール下の情報をコンパクトな一列表示に調整
  - 資金 / ファン / 知名度 / 業界評価 / 疲労 / 編成人数

## 起動方法

`index.html` をブラウザで開くと動きます。
スマホ確認は GitHub Pages 推奨です。

## GitHub Pages 更新

ZIPを解凍して、中身をリポジトリ直下へ上書きアップロードしてください。

必要ファイル：

```txt
index.html
style.css
game.js
manifest.json
sw.js
icon-192.png
icon-512.png
README.md
VERSION.txt
GITHUB_PAGES_GUIDE.md
```

古い表示が残る場合は、ゲーム上部の「最新版」ボタンでアプリキャッシュを削除して再読み込みできます。
