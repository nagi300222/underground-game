# アンダーグラウンド（仮） v0.3.20

## v0.3.20 修正内容

- 歌詞3語入力欄をスマホで見やすい1列・幅広レイアウトへ調整。
- 入力後やボタン操作後に黄色いフォーカス枠が残る問題を軽減。
- 2ターン目は未完成曲完成まで他画面操作をロックし、曲エディタへ誘導。
- 1曲目完成後に、曲作り/編曲で疲労がたまる説明ポップを追加。
- ライブ前日に疲労60%以上の場合、休憩やドリンクをすすめるポップを追加。
- スケジュール帳とライブ候補の横幅挙動を切り離し、スケジュール帳だけ横スクロールするよう調整。
- ログ履歴で、レア系は黄色、スキル/ジャンル/定番化系は紫に色分け。
- `node --check game.js` 通過。
