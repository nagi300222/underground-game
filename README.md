# アンダーグラウンド（仮） v0.3.19

## v0.3.19 修正内容

- 曲エディタを「曲作成」「編曲」の2ボタン構成へ整理。
- 曲作成側に「新曲作成」と未完成曲一覧を表示。
- 新曲作成は「作詞」「作曲」から開始できるように変更。
- 作詞は、作詞者選択 → 歌詞3語入力 → 完了の流れへ変更。
- 歌詞3語入力の各枠に🎲ランダムボタンを追加。
- 作曲は、作曲者選択 → ジャンル2つ → アレンジ1つ → 完了の流れへ変更。
- 未完成曲は、作成済み側を押せないようにし、未作成の作詞/作曲だけ進行可能に調整。
- 作詞作曲が揃った後に、曲名とテーマを選ぶ完成前確認画面へ移動。
- 「キーワード」表記をUI上では「歌詞」寄りに変更。
- 編曲側を「曲強化」「曲編集」の2ボタン構成へ整理。
- 曲強化は、ライブ等で得た曲Ptをバーで各パラメータへ振り分ける方式へ変更。
- 曲完成時に曲Pt+10を付与。
- 曲編集で、曲名・テーマ・ジャンル・アレンジ・歌詞を変更可能に。
- `node --check game.js` 通過。

## アップロード

ZIPを解凍し、中身のファイルをGitHub Pagesリポジトリ直下に上書きしてください。

`VERSION.txt` が `underground_game_v0.3.19` なら反映済みです。
