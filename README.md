# アンダーグラウンド（仮） v0.3.30

UI本格改修前の総合検証版です。

## v0.3.30 の目的

v0.3.30 は、これまで v0.3.30〜v0.3.34 に分ける予定だった以下を、UI調整前に一気に確認するための統合版です。

- ライブ/スケジュール情報整理
- 他バンド/交流/対バン導線強化
- メール/SNS/携帯導線強化
- 曲作成/曲強化テンポ調整
- ポップ/通知/ログ整理
- コピー曲評価の追加調整

## 確認してほしい流れ

1. 初ライブまで進める
2. ライブハウスイベントに参加する
3. 自主企画のワンマン/対バンを予約する
4. メールから出演通知を受ける
5. 携帯/SNSを確認する
6. 打ち上げで他バンド交流や紹介が自然に見えるか確認する
7. 曲作成・曲強化が1ターンのテンポを壊していないか確認する
8. ポップが多すぎないか確認する
9. 過去ライブ結果ログが診断ログの代わりとして使えるか確認する
