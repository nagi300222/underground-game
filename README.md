# アンダーグラウンド（仮） v0.3.13

HTML/CSS/JavaScript製のスマホ縦画面向けプロトタイプです。

## v0.3.13 修正内容

- ギターボーカル/ベースボーカルの補完キャラを追加
- コーラス枠を10ターンごとに1人追加、最大6人に拡張
- バンド最大人数と控えを撤廃し、加入メンバー全員をライブ準備に表示
- 6人以上の大所帯では維持費・交通費が重くなるよう調整
- 4人目加入時に大所帯コストの説明ポップを追加
- エンディング後に続行した際、通常ライブ予約候補が復旧するよう修正

## アップロード方法

ZIPを解凍して、中身の `index.html`, `game.js`, `style.css`, `manifest.json`, `sw.js`, `README.md`, `VERSION.txt` をGitHubリポジトリ直下へ上書きしてください。

`VERSION.txt` が `underground_game_v0.3.13` になっていれば反映済みです。

古い表示が残る場合は、ゲーム内の「最新版を読み込む」を押してください。v0.3.13では `sw.js` のキャッシュ名も更新しています。


## v0.3.11 追加メモ

- ライブ中、5曲目が二重判定される問題を修正。
- ライブ準備でセトリ内の曲順を上下移動可能に。
- バンドポジションの簡易ステータス表示を追加。
- バンド構成内と控えを直接入れ替える導線を追加。
- ショップ/ライブ予約などの確認ポップは中央合わせ・下部ボタン揃え寄りに調整。
- 物販は種類別に仕入れ数を選ぶ方式へ変更。キャパ・客層・ライブ評価・曲認知で売れやすさが変動。売れ残りは仕入れ値の70%で買い取り。
