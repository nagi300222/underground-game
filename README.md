# アンダーグラウンド（仮）

スマホブラウザ向けバンド育成SLG。

## 現行版

`v0.4.3`

動作保証は iPhone Safari のみ（Android 実機未検証）

## GitHub Pages 公開に必要なルートファイル

```text
index.html
game.js
style.css
sw.js
VERSION.txt
manifest.json
.nojekyll
README.md
```

監査資料・CHANGELOG・Fable5資料は `docs/` 以下に整理しています。

## メモ

`.nojekyll` は GitHub Pages のJekyll処理を止めるために置いています。削除しないでください。
