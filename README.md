# アンダーグラウンド（仮） v0.3.17

## v0.3.17 修正内容

- ドリンク弱体化をいったん撤回。価格5,000円、疲労-25へ戻しました。
- スケジュール画面をスマホ幅に収まりやすく調整し、予約操作後に拡大しにくいよう入力/ボタン/タップまわりを調整しました。
- 曲作りUIを強化しました。
  - 作詞/作曲Lv表示
  - 疲労軽減/疲労なし確率の表示
  - 作詞3語入力UIの改善
  - 各入力欄に🎲ランダムボタン追加
  - 作詞者/作曲者の補正表示を強化
  - 作詞/作曲Lvアップポップ
  - 疲労なし制作ポップ
  - 未完成曲一覧を見やすく調整
- 編曲モードを追加しました。
  - 曲強化とは別に、ジャンル/アレンジ/テーマ/歌詞キーワードを編集可能
  - ライブで得る曲Ptを使って指定パラメータを伸ばせる
  - 編曲後は棒グラフつきポップで変化を表示
- 曲選び支援を追加しました。
  - 曲一覧ソート：総合値/新曲/使用回数/マンネリ危険/定番曲
  - 曲カードにおすすめタグを表示
  - ライブ準備でセトリ自動ヒントを表示
  - 曲選択画面は客層相性を加味した順に表示
- ポップ演出を追加/強化しました。
  - 作詞Lv/作曲Lvアップ
  - ひらめき制作
  - 定番曲化
- 練習成果の見える化を強化しました。
  - 上昇前→上昇後を表示
  - 簡易バー表示
  - 疲労による効率低下の説明を追加

## アップロード

ZIPを解凍して、中身の `index.html`, `game.js`, `style.css`, `manifest.json`, `sw.js`, `README.md`, `VERSION.txt` をGitHubリポジトリ直下へ上書きしてください。

`VERSION.txt` が `underground_game_v0.3.17` になっていれば反映済みです。

古い表示が残る場合は、ゲーム内の「最新版を読み込む」を押してください。v0.3.17では `sw.js` のキャッシュ名も更新しています。
