# アンダーグラウンド（仮） v0.3.2

スマホ縦画面向けのバンド育成プロトタイプです。v0.3.2では、曲作りのジャンル・テーマ・アレンジ周りを整理しました。

## 主な変更

- テーマを少数の大テーマに整理
  - 初期：恋愛 / 失恋 / 友情 / 応援 / 日常 / 青春 / 反抗 / 孤独 / 自分 / 居場所
  - 中盤以降、焦燥感・劣等感・個性などを解放予定
- メインジャンルを段階解放式に整理
  - 前半：ロック / ポップ / パンク / メタル
  - 中盤：ヒップホップ / ミクスチャー / エモ / オルタナ / エレクトロ
  - 後半：ジャズ / クラシック / プログレ / シューゲイザー / ポストロック
- サブジャンルはメインジャンル2つから抽選で派生
- 前半サブジャンル表を仮実装
  - ギターロック / ハードロック / ガレージロック / オルタナロック / バラードロック
  - ギターポップ / パワーポップ / アニソンロック / シティポップ / バラードポップ
  - ポップパンク / メロディックパンク / 青春パンク / スカパンク / ハードコア / ガレージパンク
  - ヘヴィメタル / スピードメタル / メロディックメタル / メタルコア / スクリーモ
- 初期レアジャンルを追加
  - デジロック / ダンスロック / グランジ / ポストハードコア / ニューメタル
- ピアノ主導限定レアとして「ピアノロック」を追加
- アレンジに必要楽器条件を追加
  - ツインギターはギター2人、DJミックスはDJ、コーラス重視は別コーラスなど
- アレンジ状態を「有効 / 不完全 / 無効」で表示
- キーワード自由入力と簡易辞書判定を追加
- 曲作りをしていない状態で行動すると確認ポップアップを表示

## 起動方法

PCなら `index.html` をブラウザで開くだけでも動きます。スマホ確認はGitHub Pages推奨です。

## GitHub Pages更新

ZIPを解凍して、中身のファイルをリポジトリ直下に上書きアップロードしてください。ZIP自体ではなく、`index.html` / `game.js` / `style.css` などをアップロードします。

## 更新時の注意

古い表示が残る場合は、ゲーム内の「最新版を読み込む」を押してください。v0.3.2では `sw.js` のキャッシュ名も更新しています。


## v0.3.2 修正

- 初回チュートリアルの「まずは曲作り」後に、作詞・作曲ボタンを押しても再度ポップアップが出て進めない不具合を修正。
- 新曲100%完成時に止まる可能性があった未定義変数を修正。
