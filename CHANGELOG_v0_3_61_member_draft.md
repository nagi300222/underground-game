# v0.3.61-member-draft-fix1

## 追加内容

- プレイヤーバンド加入候補を、他バンドキャラと被らない汎用カタカナ名の新データへ差し替え。
- デフォルト名は苗字のみ・名前のみ。加入時にプレイヤーが表示名を変更可能。
- `Gt/Vo` と `Ba/Vo` を正式な加入候補担当として追加済み。ライブ準備では既存の `ギターボーカル` / `ベースボーカル` として扱える。
- 初期16人 / 中盤12人 / 後半8人の計36人を `MEMBER_DATABASE` として統合。
- `MEMBER_GROWTH_TYPE_DATABASE` を追加。今回は表示・設計メモ用で、成長式への本格反映は後続。
- 加入条件を `joinCondition` に追加し、中盤・後半候補が序盤に出すぎないように調整。
- 名前付きサポート候補8人を `SUPPORT_MEMBER_DATABASE` として追加。
- 旧候補プールに残っていた他バンド代表系の仮キャラは、新規データで上書き。旧セーブの未加入候補・候補プールは現行IDだけ残す。

## 確認

- 候補ID重複なし、候補名と他バンド代表名の直接被りなしを確認。
