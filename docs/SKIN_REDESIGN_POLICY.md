# スキン刷新方針 v1.1（D-1b採用）

発注日: 2026-07-17 ／ 適用開始: 段階1（PR-A: CSS基盤＋フォント資産）

## 裁定（確定値）

1. **書体**: OFLライセンスのコンデンス書体を自前woff2同梱。採用: Barlow Condensed SemiBold（`@fontsource/barlow-condensed` 5.2.8 由来）。ラテン印字可能域（U+0020–007E、英数字＋基本約物）でサブセット化し `fonts/barlow-condensed-semibold-subset.woff2`（約8.8KB）として同梱。`fonts/OFL.txt` にライセンス全文を同梱。和文はシステムフォント続投（`font-family` の第一候補に条件付きで指定し、日本語グリフはフォントに存在しないため自動的にシステムフォントへフォールバックする挙動を利用）。
2. **アイコン/記号**: `▌ ▶ ■` 等の記号グリフはBarlow Condensedに存在しないことを確認済み。段階2（game.js表層）で絵文字を記号＋異体字セレクタ（U+FE0E）へ置換する際、形が足りない記号はインラインSVGで補う。
3. **疲労発光例外**: 不採用。発光は導線専用（主アクションボタン・活性タブ）の2箇所固定。

## トークン

| 役割 | 値 | 用途 |
|---|---|---|
| BG | `#0B0B0D` | 全面基調のフラット黒 |
| INK | `#EAE6DD` | 本文・罫線・非活性ボタン枠 |
| ACC | `#FF4A00` | 唯一のアクセント（主ボタン・活性タブの発光のみ） |
| DIM | `#6E695F` | ラベル・キャプション限定 |
| RULE | `#33312C` | 弱い区切り罫 |

実測コントラスト比（対BG、WCAG相対輝度式）: INK 15.79:1／ACC 5.84:1／DIM 3.60:1。

## 組版・形状・発光

角丸ゼロ（`border-radius: 0` に統一）。枠線・バーは原則2px実線、弱区切りのみ1px。見出し（`h1`–`h6`）とボタン（`button`, `select`）はコンデンス書体＋大文字＋広め字間（見出し0.08em、ボタン0.2em）。数値・本文中のラテン文字は `body` の `font-family` 先頭にコンデンス書体を指定することで全画面に自動適用（和文は非対応グリフのため自動フォールバック）。ゲージは枠2px（INK）＋斜線塗り（`repeating-linear-gradient(-45deg, ACC 0 4px, transparent 4px 8px)`）。ACCのベタ塗り面は不使用——ACCは線・文字・光のみ。

発光は以下2箇所に固定:
- 主アクションボタン（`.home-main-action`）: `border: 2px solid var(--acc)` ＋ `box-shadow: 0 0 14px var(--acc), inset 0 0 14px rgba(255,74,0,.22)` ＋ 文字ACC。
- 活性タブ（`.tabbar .tabBtn.active`）: 文字ACC＋グロー（`text-shadow`）＋ `▌` マーカー（`::after` の `content: "▌"`）。

## ネガティブ仕様の遵守状況

グラデーション（斜線塗りゲージ以外は全廃、`var(--bg)` へ平坦化）／角丸（全数ゼロ化）／ドロップシャドウ（`box-shadow`/`text-shadow`/`filter: drop-shadow` を全数 `none` 化、上記2箇所の発光のみ例外）／絵文字（style.css内に絵文字content指定なしを確認済み）／紙・ノイズ質感（グラデーション/半透明レイヤーの平坦化により実質消滅）／走査線（該当なし）／等幅計器風数字（`font-variant-numeric: tabular-nums` を全数除去）／ACCベタ塗り面（背景としてのACC使用は皆無、線・文字・光のみ）。

## 実装上の判断メモ（段階1）

- 旧トークン体系は `:root` の多層変数（`--v041-*` / `--v41-*` / `--v043e-*` 含む）を経由していたため、各変数定義をBG/INK/DIM/RULEへ再マッピングする方式で全数置換した。単一変数が背景用途と前景（枠線）用途の両方で使われていた箇所（`--v41-red` の通知バッジ背景など）は、変数定義とは別に使用箇所を直接上書きして矛盾を回避した。
- モーダル背景の全画面スクリム（`.story-backdrop` / `.modal-backdrop`）は視認性維持のためBGを基調とした半透明（`rgba(11, 11, 13, .82)` 相当）を許容し、平坦化の対象外とした。
- 1px/2px の使い分けは「既存の枠線幅が1pxだったもの＝弱区切り」という機械的な判定を基本方針としつつ、モーダル外枠やゲージ枠などコンポーネントの主要境界と判断した箇所は2px（INK）に統一した。
