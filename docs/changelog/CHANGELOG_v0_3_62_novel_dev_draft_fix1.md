# CHANGELOG v0.3.62-novel-dev-draft-fix1

対象：スマホブラウザ向けバンド育成SLG『アンダーグラウンド（仮）』
土台：v0.3.62-novel-dev-draft

## 目的

v0.3.62で追加された簡易ノベルイベントUIとDEVモードを検証し、進行不能・接続漏れ・DEV不具合を修正。
優先4イベント（Triple Arrows招待 / CARBONS初対バン / SHELTER遭遇 / 汎用加入）を実際のゲーム導線へ接続した。

## 修正した問題

1. **ノベルOFF時に初回打ち上げが無表示になる問題**
   ENABLE_NOVEL_EVENTS=false かつバンドシステムON時、初回打ち上げでストーリーもポップアップも
   出ないまま進行待ちになっていた。ノベル再生に失敗した場合はv0.3.60相当のポップアップへ
   フォールバックするよう修正。

2. **DEV「加入候補を任意で出す」が条件付き候補で機能しない問題**
   normalizeCandidatePoolsForCurrentData が加入条件未達の候補を応募リストから毎レンダリング
   除去するため、後半候補（ツキノ等）をDEVで提示しても即消えていた。DEV提示候補に devForced
   フラグを付与し、条件ゲートを免除するよう修正。

3. **ノベルイベントの多重起動・進行競合の予防**
   - startStoryEvent：再生中の二重起動を拒否（呼び出し側はポップアップへフォールバック）
   - hasBlockingPopupBeforeTurnAdvance：ノベル再生中はターン進行をブロック

## イベント接続（新規）

- **Triple Arrowsからの誘い**：6T以降の招待メールイベント発火時、ポップアップの代わりに
  ノベルシーンを再生（メール送付・オファー生成・図鑑更新はそのまま）。結果画面に開催ターンを表示。
- **CARBONS初対バン**：CARBONSとの初対バン記録時にノベルシーンを再生。
  ライブ結果→打ち上げの後に表示される（オーバーレイ優先度による）。
  大成功時の「ダイヤの原石」導線（打ち上げポップアップ）とは独立・競合なし。
- **SHELTER遭遇**：ライブ後の遭遇イベント発火時にノベルシーンを再生。「ロック魂」導線は従来通り。
- **汎用加入イベント**：joinApplicantById で加入確定時にノベルシーンを再生
  （名前変更・仮加入/本加入・36人データ・セーブロードは無変更）。ノベルOFF時は従来ポップアップ。
