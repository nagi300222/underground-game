# CHANGELOG v0.3.64-growth-type-draft

対象：スマホブラウザ向けバンド育成SLG『アンダーグラウンド（仮）』
土台：v0.3.63-novel-events-draft-fix1

## 今回の目的

v0.3.61で加入候補に保持していた `growthType` を、まずは小補正として実際の成長処理へ反映する。
バランス本調整は v0.3.66-balance-draft へ回し、今回は「成長タイプが数値へ効く」ことを主目的にした。

## 追加・変更

- `MEMBER_GROWTH_TYPE_DATABASE` を実効果向けに拡張。
  - 技術型 / センス型 / メンタル型 / カリスマ型 / リズム型 / 協調性型 / 体力型 / 知識型 / 万能型 / 安定型 / 序盤型 / 爆発型 / 晩成型。
- 主人公に `growthType: "安定型"` を設定。
- `growthType` 未設定メンバーの旧セーブ安全対応を追加。
  - 既存メンバー / 加入候補 / 応募者 / 主人公を normalize 時に補正。
  - 候補DBに同IDがあればDB側の成長タイプを復元。なければ安定型へ。
- 練習時の成長補正を追加。
  - 技術・リズム上昇へ成長タイプ倍率を反映。
  - 重点ステータスが低確率で追加成長。
  - 楽器Lv上昇にも成長曲線倍率を小さく反映。
- ライブ後成長補正を追加。
  - メンタル・カリスマ上昇へ成長タイプ倍率を反映。
  - S/A/B評価時、重点ステータスが追加成長することがある。
  - 楽器Lv上昇にも成長曲線倍率を小さく反映。
- 練習結果表示を拡張。
  - 技術・リズム固定表示から、実際に伸びたステータス一覧表示へ変更。
  - 成長タイプ名も表示。
- メンバーカード表示を更新。
  - 「表示のみ。実成長倍率は今後調整予定。」を削除。
  - 成長タイプの効果メモを表示。
- DEV画面に「成長タイプ確認」カードを追加。

## 補正方針

- 単一/複数特化型：重点ステータスに小補正。
- 万能型：全体に薄い補正。
- 安定型：大きく尖らないが安定して伸びる。
- 序盤型/体力型：序盤に強め、後半は控えめ。
- 晩成型：序盤控えめ、35T以降に伸びる。
- 爆発型：練習/ライブで上振れすることがあるが、わずかにブレもある。
