# CHANGELOG v0.3.69-action-mini-events-draft

## 目的

休憩以外の通常行動に、低頻度・条件付きのサクセス風ミニイベントを追加する。
既存の行動後ランダムイベントを、30T UNDER FES前後とステータス条件に連動する形へ整理した。

## 変更内容

- `maybeGenerateCommandEvent` を条件付き発生方式へ更新。
- 休憩ではミニイベントを発生させない。
- 基本発生率を15%程度に抑制。
- バンド/個人ステータス、25〜29T、31T以降、GRAND準備期で発生率が変化。
- 30T前後でイベント文面を切り替える。
- 対象行動：
  - 練習
  - 宣伝
  - 募集
  - バイト
  - 会話
- 効果候補：
  - ステータスUP
  - 知名度/ファン/業界評価UP
  - 資金少量獲得
  - 曲の軽い強化
  - freeSongcraftCharge付与
  - bookingInviteBoostTurns / promoRecruitTurns補助

## 重要維持

- 作詞/作曲/曲完成/曲強化ではターン進行しない仕様を維持。
- 通常行動後のターン進行予約は従来どおり。
- 行動結果モーダル内にイベントを表示する方式を維持。
- 休憩はイベントなし。
- v0.3.68の打ち上げイベントを維持。

## 注意

- 作詞/作曲系のミニイベント本格対応は、ターン進行ルール保護のため今回は未実装。
- 台本は仮文。今後差し替え予定。
- 効果量は小さめの仮値。Fable5外部監査でバランス相談対象。
