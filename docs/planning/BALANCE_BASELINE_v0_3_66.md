## 3. キャラステータス基準

キャラ個人ステータス：体力 / 技術 / 知識 / センス / メンタル / 協調性 / リズム / カリスマ。

### 加入直後の低め

- 10〜24：明確な弱点
- 25〜39：序盤相応

### 標準

- 40〜54：普通に戦力
- 55〜69：得意分野

### 高い

- 70〜84：強い個性
- 85〜99：トップ級・終盤級

## 4. 成長タイプの想定

v0.3.64時点では小補正。

- 技術型：練習・ライブ後に技術寄り
- センス型：作詞作曲・表現寄り
- メンタル型：ライブ後・疲労高めでも崩れにくい方向
- カリスマ型：ライブ後・集客印象寄り
- リズム型：練習効率寄り
- 協調性型：バンド全体・交流イベントと相性を持たせる候補
- 体力型：疲労耐性候補
- 知識型：曲作り・戦略寄り候補
- 万能型：全体小補正
- 安定型：ブレ抑制
- 序盤型：前半寄り
- 晩成型：25T以降に伸びやすい
- 爆発型：低確率上振れ

注意：v0.3.66では、まだ本格調整ではなく「効きすぎていないか」を見る段階。
