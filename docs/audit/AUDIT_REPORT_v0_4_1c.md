## 既知の残課題（ブロッカーでない違和感）
1. 紙面パネル（準備・図鑑・貼り紙）上の一部子要素は配色を個別上書きしており、
   未知の子クラスが混ざると暗地文字が紙に載る可能性。実機で紙面3画面を目視確認。
2. 図鑑フライヤーの微傾き（±0.4deg）は端末によりにじむ場合あり。気になれば transform 2行削除で無効化可。
3. text-wrap:balance/pretty は iOS17.4+/Android Chrome前提の漸進強化。旧端末では従来折返し（実害なし）。
4. ライブ結果のランクスタンプは既存 .live-result-modal.rank-X フックに依存。ランク表記変更時は要追随。
